avimerge

avimerge ist im transcode Paket enthalten. Die Installation dauert sehr lange, da MacOS anscheinend viele benötigten Libraries nicht standardmäßig enthalten. 

    $ sudo port install transcode

Mehrere Videoparts in eine Datei mergen 

    $ avimerge -o OutputFile.avi -i Input1.avi Input2.avi

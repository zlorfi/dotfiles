Apple Mail

Mailalias

  • Mail.app schließen
  • ~/Library/Mail/V2/MailData/Accounts.plist öffnen
  • Eintrag MailAccounts aufklappen
  • den richtigen Ornder (0, 1, 2, etc) rausfinden indem man auf die Einträge achtet
  • Ordner auswählen und Add Item anklicken, mit dem Namen EmailAliases mit Typ Array
  • diesen wiederum anklicken und mit Add Item einen neuen Eintrag Item 0 erstellen vom Typ Dictionary
  • auswählen neuen Key mit alias vom Typ String und der gewünschten Adresse
  • im Item 0 neuen Eintrag mit name als Item String als Typ und dem gewünschtem Namen für den Alias
  • abspeichern, fertig.
